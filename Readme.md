# CodeAlpha AI Chatbot

This project was developed as part of the CodeAlpha Java Programming Internship.

## Project Description
AI Chatbot is a simple Java-based chatbot that interacts with users through the console.  
The chatbot responds to basic questions using rule-based logic.

## Features
- Interactive chatbot conversation
- Rule-based responses
- Simple NLP using keywords
- Console-based interface
- Exit option using "bye"

## Technologies Used
- Java
- OOP Concepts
- Scanner Class

## How to Run

1. Compile the program:
```bash
javac AIChatbot.java

2.Run the program:
```Bash
java AIChatbot
